# multichat-stickers-extension
A Chrome extension that adds multichat stickers to YouTube and Twitch chat.

![](https://misakamibot.ru/multichat/imgs/extension-demo.png)

## Installation
1. [Download](https://misakamibot.ru/multichat/files/multichat-stickers-extension.crx) the extension
2. Open `chrome://extensions/` or `browser://extensions/`
3. Activate the switch "Developer mode" in the upper right corner
4. Drag and drop .crx file on the page
5. If the browser says "Chrome can't verify where this extension comes from", click "Keep this extension."
![](https://misakamibot.ru/multichat/imgs/extension-security.png)
